Executive Summary
AEGIS detected a multi-stage attack chain originating from spearphishing delivery on host DVWS-PC01. The attack leveraged a macro-enabled Excel attachment to initiate a LOLBin execution chain: EXCEL.EXE → cmd.exe → mshta.exe → powershell.exe (encoded) → certutil.exe (decode). The decoded payload was injected into a legitimate svchost.exe process (PID 4712) via NtCreateThreadEx hollowing. The compromised process immediately attempted LSASS credential dumping and established DNS-based command-and-control communication using domain generation algorithm patterns at 30-second beacon intervals. Lateral movement to domain controller DVWS-DC02 was attempted via SMB with harvested credentials. AEGIS Guardian contained the endpoint within 1.8 seconds, blocking exfiltration. The attack pattern matches Scattered Spider TTPs at 84% confidence with Volt Typhoon LOTL techniques at 67%.
